Partially ship an order in woocommerce.2. you can set custom order status for partially shipped orders.3. Ship item quantity as per availability.4. Auto send email notification when item is partially shipped.5. Retrieve/Update partially shipped data by Curl request API.6. Modify Status label using WordPress filter available in code.7. Custom Status label.8. Quick pro Support.9. Customer can see the shipped items on order detail page.== Installation ==1. Simply install and activate the plugin.2. Now you can see Shipment button and icon on order edit page.3. you can set item shipment there.4. Partial Shipment Settings is under <strong>Woocommerce >> Settings >> Partial Shipment tab.</strong>5. Curl Req Example is available there in Curl-Req-Example directory.== Frequently Asked Questions === having issue or need help with Woocommerce Partial Shipment plugin ? =Contact us at https://wooexperts.com/contact-us/ with your Order details.= How to set up CURL Req API =Check the Curl Req Example available inside directory "/Curl-Req-Example".Make sure the API is enabled in settings Woocommerce >> Settings >> Partial Shipment tab.== Changelog === 1.6 - 31/03/2022 =* Fix - Updater issue fixed= 1.5 - 29/01/2022 =* Fix - WC 6.1 compatibility fix= 1.4 - 28/08/2021 =* Fix - php version 8.0 compatibility= 1.3 - 30/03/2021 =* Fix - Sku added in stored data.= 1.2 - 22/03/2021 =* Fix - Curl-Req get set data.= 1.1 - 18/02/2021 =* Fix - Fixed error during plugin activation.= 1.0 - 13/02/2021 =* Initial release.
