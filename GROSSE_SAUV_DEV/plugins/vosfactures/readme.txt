Tested up to: 6.0
Stable tag: 6.0
Requires PHP: 5.6

The plugin allows you to issue customized invoices in PDF for your orders by integrating your WooCommerce store with the multifunction invoicing software VosFactures.

== Description ==

** Features **

- Manual & Automatic issuing of invoices on VosFactures after placing an order/changing order status;
- Choose when invoices should be generated : automatically after order is created, completed, processing, or only manually;
- Customize invoice appearance and content thanks to numerous options from your VosFactures account - available sample templates and ability to create your own template;
- Download PDF of invoices from your WooCommerce backend;
- Email invoices to your clients automatically or manually;
- Issue easily correction invoices if needed;
- Centralize & analyze on your VosFactures account all your invoices & related data of your WooC. and direct sales.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly (WordPress Admin > Plugins > Add New).
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce->Vosfactures screen to fill the API Token with prefix of your VosFactures account (to be generated from Settings -> settings -> Integration). If it is correct, the rest of the settings will appear.
